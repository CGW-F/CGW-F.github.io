欧几里得算法与欧几里得扩展算法

欧几里得算法

gcd(a,b) :a和b的最大公因子（公约数），下面同记为(a,b)，a，b∈Z

1.定理：(a,b) = (b,a mod b)**

推导过程：

∵ a，b∈Z，

∴ 有a = kb + r , k,r∈Z ①

则 a mod b = r ②

联合①②： a mod b = a-kb ③

设g = gcd（a，b）

则：a%g = 0（a mod g=0），b%g = 0；即a%g = b%g

可以推出：(a - b)%g = 0 ④

（a、b都是g的倍数关系，所以a和b的差值也是最大公因子g的倍数关系）

结合③和④

推出： （a mod b）%g = 0

所以g是a，b，（a mod b）的公因子

∴ (a,b) = (b,a mod b)

2.欧几里得算法（辗转相除法）

第一步：（a0，b0）

第二步：（a1，b1），a1= b0, b1 = a0%b0

……

第n步：（an，bn），当bn = 0时，an就是（a，b）最大公约数

扩展欧几里得算法

1.求逆元原理

推导过程：

设（a，b）是a，b的最大公约数，则有a.x1 = k1(a,b), b.y1 = k2(a,b) ,k1,k2∈Z “.”是乘法

可以推导出：a.x1 + b.y1 = k3(a,b) ，k3 = k1+k2，k3∈Z

所以必然存在

​ ax + by = （a，b ）

当（a，b）=1时，a，b互质（又称为互为素数），得：

​ ax + by =1 （x是a的逆元，y是b的逆元）

所以有：a*X = bY+1 （正负号写不写无所谓，因为它对公式没什么意义，Y可正可负）

​ X = （bY+1）/a

​ Y = （aX+1）/a

2.扩展的欧几里得算法

由欧几里得算法得：

​ （a，b） = （b，a mod b） ①

​ ∵ ax + by = (a,b) ②

联合①②得：

​ b.x1 +（a mod b）.y1 = (a,b) ③

​ a mob b = a%b = a - (a/b).b ④ “/“是除法

联合③④

​ gcd = （a，b） = b.x1+[a - (a/b).b].y1

​ =b.x1 + a.y1 - (a/b).b.y1

​ =a.y1 + b.[x1 - (a/b).y1]

对比：(a,b) = ax + by

可以推导出：

​ x = y1

​ y = x1 - (a/b).y1

//aX+bY=gcd；x是a的逆元，Y是b的逆元；ans（gcd）是最大公约数  
LL e_gcd(LL a,LL b,LL &x,LL &y)  
{  
    if(b==0)  
    {  
        x=1;  
        y=0;  
        return a;  
    }  
    LL ans=e_gcd(b,a%b,x,y);  
    LL temp=x;  
    x=y;  
    y=temp-a/b*y;  
    return ans;  
}  

LL cal(LL a,LL b,LL c)  
{  
    LL x,y;  
    LL gcd=e_gcd(a,b,x,y);  
    if(c%gcd!=0){ 
        printf("没有逆元");
        return -1;  //判断是否有逆元or两个数（a、b）是否质，即gcd最大公约数是否是1     
        }
    x*=c/gcd;  
    b/=gcd;  
    if(b<0) b=-b;  
    printf("x的逆元=%ld,y的逆元=%ld\n",x,y); 
    //这里求得的逆元可能为负数，再求mod转回正数即可 
    LL X=x%b;

    return X;  
}

SM3哈希算法

一、算法概述

​ SM3哈希算法是我国国产密码算法。除了SM3加密算法外，我国还有SM2、SM4、SM9等加密算法。SM3算法的结构与SHA-512的结构有些相似，采用了Merkle-Damgard结构（简称M-D结构），这是一种压缩迭代结构，是构造压缩函数的核心；有压缩必然有碰撞，这是M-D结构的主要缺点；为了“完全”避免哈希碰撞，第三代SHA采取了海绵结构（Sponge结构）。

椭圆曲线加法定义

说明：如果线上3个点位于同一直线上，那么它们的和为O

  1. O为加法单元，椭圆上任意一个点P，都有P+O=P；它运算上类似于0，但它意义不是一个简单的点的定义，无穷远点也是O。

  2. 加法逆元定义：P=（x，y），逆元p’=（x，-y）；关于X轴对称。这是因为P1、P2的连线延长到无穷远时，得到的是椭圆曲线上的另外一点O，即P1、P2、O三点共线，所以P1+P2+O=O，P1+P2=O，即P2=-P1。（如图b曲线，当P趋向无穷远时，P1、P2、无穷远点这3个点就共线了）

  3. 如图a，设Q和R是椭圆上的x坐标不同的两点。Q+R的定义：画一条Q和R的直线，与曲线交于P1（这交点是唯一的，除非所做的直线是Q和R的切线，此时P1=Q=R），由于Q、R、P1这3点共线，则Q+R+P1=O，得Q+R=-P1。

  4. 点Q的倍数（倍乘）定义：在Q点做椭圆曲线的一条切线，设切线与椭圆曲线交于点S，则2Q=Q+Q=-S。

  5. 椭圆曲线加法符合的运算法则：交换律、结合律

RSA加密算法及其数字签名

1、密钥产生

  • 选取两个大素数p和q

  • 计算n=p*q，φ(n)=(p-1)(q-1)，即φ(n)是n的欧拉函数值

  • 选取随机数e，1<e<φ(n)，且 gcd(φ(n)，e)=1

  • 计算d：

    ​ d*e=1 mod(φ(n))

  • 公钥{e,n}，私钥{d,n}

说明：

​ a.gcd(φ(n)，e)=1：最大公因子/公约数为1，即e和φ(n)互素

​ b.d*e=1 mod(φ(n))：由逆元定义得知，e和d互为逆元

​ c.到2021年为止，n的有效安全二进制位数为2048位（十进制大概608-609位），但1024位的也还有在使用
